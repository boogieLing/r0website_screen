# UShouldKnow Experience Icons

Put your custom PNG icons in this folder.

Expected file names:
- openvision.png
- tencent.png
- msra.png
- deeproute.png

Recommended spec:
- format: png
- size: 64x64 or 128x128
- transparent background preferred

Behavior:
- If file exists: show PNG
- If file missing: fallback emoji will be shown automatically

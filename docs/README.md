# 文档总览

本目录用于集中存放项目的 Markdown 文档，按用途进行分类：

- `system/`：系统设计与规格说明  
  - `ABSTRACT_TAB_SYSTEM.md`：抽象 Tab 系统设计说明  
  - `SOMNIUM_NEXUS_IMPLEMENTATION.md`：Somnium Nexus 实现细节  
  - `SOMNIUM_NEXUS_OPTIMIZATION.md`：Somnium Nexus 优化策略  
  - `SOMNIUM_NEXUS_VERIFICATION.md`、`SOMNIUM_NEXUS_FINAL_VERIFICATION.md`：Somnium Nexus 验证与最终验证记录

- `tests/`：交互与样式相关的测试记录  
  - `test-animation-hover-fix.md`、`test-hover-fix.md`、`test-first-hover-fix.md`：Hover 与动效相关测试说明  
  - `test-scrollbar-fix.md`：滚动条问题测试与修复记录  
  - `test-maincontent-stability.md`：主内容区域稳定性测试  
  - `test-interaction.md`：整体交互测试记录

- `agents/`：智能助手与协作说明  
  - `CLAUDE.md`：Claude 使用与协作说明（针对特定助手）  
  - 根目录 `AGENTS.md`：针对本仓库所有智能助手的统一指南

- 根目录文档（未移动）  
  - `README.md`：项目总览与快速开始  

其他说明文档包括：  
- `CONSOLE_COMMANDS*.md`：控制台指令说明与示例  
- `DATA_MIGRATION.md`：数据迁移相关流程与注意事项
